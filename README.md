# Random Password Generator

## Автор
Иванов Иван

## Описание
Простое приложение для генерации случайных паролей с настройкой параметров и сохранением истории.

## Требования
Python 3.x

## Установка
1. Клонируйте репозиторий:
   git clone <ссылка>
2. Установите зависимости:
   pip install -r requirements.txt

## Использование
1. Запустите main.py.
2. Настройте параметры и сгенерируйте пароль.
3. История сохраняется в history.json.

## Примеры тестов
- Генерация пароля длиной 12 символов с цифрами и буквами.
- Генерация пароля только со спецсимволами.
